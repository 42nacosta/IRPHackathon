About This Project

This web app is a IRP group project built for the Career Chase Hackathon, built by Nathaniel Acosta and Evan Acosta. It follows the Django MVT (Model–View–Template) architecture and targets data analysts that want intuitive and effective data analysis of US Soccer matches.

What You Can Do
  • Enter match data and view a visualized replay of the game.
  • Use the controls to play, pause, and scroll through the contents of the replay.
  • Take a snippet of the replay and view important data.
In Scope for This Hackathon
  • Implement the ability to load a replay.
  • Take a snippet of replay and create/share data visualization graphs.
Architecture at a Glance (MVT)
  • Models: Matches to store data for easy access.
  • Views: Readable and intuitive feedback and data visualization.
  • Templates: Bootstrap-styled pages extending a shared base.html.
Team

Built by Nathaniel Acosta and Evan Acosta. We are a duo of motivated CS students from Savannah, Georgia. This project is built using Django and streamlit functionality to display the visual replay, and SkillCorner for data and data visualization.
